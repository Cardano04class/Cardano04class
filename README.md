## 🏆 Featured Projects

Here is a curated selection of some of my most challenging and rewarding projects:

### 🎮 [ft_transcendence](https://github.com/Cardano04class/ft_transcendence)
> **Modern Full-Stack Multiplayer Web App**
- **Overview:** The final capstone project at 42. A single-page web application featuring real-time multiplayer Pong games, match history, user matchmaking, customization options, and live interactive channels.
- **Key Features:** User accounts with OAuth 2.0 (42 Intra) & Two-Factor Authentication (2FA), matchmaking lobby, real-time live chat with channels, game history, and leaderboards.
- **Tech Stack:** TypeScript, Next.js, WebSockets, PostgreSQL, Docker, Docker Compose.

### 🐳 [Inception](https://github.com/Cardano04class/Inception)
> **Advanced DevOps & Infrastructure Virtualization**
- **Overview:** Fully dockerized a multi-container system architecture, deploying various services within secure, custom-built environments.
- **Key Features:** Configured a secure Nginx server with TLSv1.3/TLSv1.2, MariaDB database, and WordPress engine. Every service runs in its own dedicated, isolated Docker container built from Alpine Linux scratch images.
- **Tech Stack:** Docker, Docker Compose, Nginx, MariaDB, WordPress, Shell scripting.

### 🐚 [minishell](https://github.com/Cardano04class/minishell)
> **Unix Systems Programming**
- **Overview:** Co-built a lightweight command-line interpreter (shell) in C, recreating a core operating system interface.
- **Key Features:** Implemented pipes (`|`), input/output redirections (`<`, `>`, `>>`, `<<`), environment variable expansion, signal handling (`Ctrl-C`, `Ctrl-\`, `Ctrl-D`), and built-in utilities like `cd`, `echo`, `export`, and `unset`.
- **Tech Stack:** C, Unix System Calls, POSIX, Shell Scripting.

### 📊 [Push_swap](https://github.com/Cardano04class/Push_swap)
> **Algorithms & Complexity Optimization**
- **Overview:** An extremely optimized sorting algorithm project designed to sort numbers using a limited set of stack operations.
- **Key Features:** Achieved high efficiency by implementing custom sorting mechanics (using optimized chunk-sorting and coordinate-compression techniques) to minimize instruction count.
- **Tech Stack:** C, Algorithms, Big-O Complexity analysis.
